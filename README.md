# text-toolkit

多用途なウェブベースのテキスト整形・変換ツールです。

## 主な機能

- **💻 コードブロック化**: 入力テキストを各種プログラミング言語のコードブロック（Markdown、JavaScript、Python等）に自動または手動で変換します。
- **✍️ テキスト整形**: 生成AI（Geminiなど）が出力したMarkdownの調整、括弧のネスト自動調整、箇条書きや番号付きリストへの一括変換に対応しています。
- **🔄 空白・改行の変換**: 行頭・行末の空白削除、全角・半角スペースの相互変換、改行からスラッシュやスペースへの変換などを行えます。
- **🔐 エンコード・デコード**: Base64やURLのエンコード・デコード処理を手軽に実行できます。

## 特徴

- 「貼り付け」「変換」「コピー」をシームレスに行える直感的なレイアウト設計
- ダークモードを搭載し、スマートフォンとPCの双方に最適化されたレスポンシブなユーザーインターフェース
- Service Workerによるキャッシュ制御を行い、オフライン環境での利用にも配慮（PWA対応）

## ⚙️ 開発情報

### キャッシュのバージョン管理ルール

Service Worker (`sw.js`) で使用するキャッシュ名 (`CACHE_NAME`) は、以下の規則で運用しています。

`text-toolkit-YYYY.MM-rN`

- **YYYY** = 西暦 (例: 2026)
- **MM**   = 月 (01〜12)
- **rN**   = その月のリリース回数 (例: r1, r2...)

例: `text-toolkit-2026.07-r1`

リリースやファイルを更新する際は、このバージョン文字列を更新することで、ブラウザの古いキャッシュを安全に破棄・新しいファイルを適用できます。
